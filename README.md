## 💻 Projeto
Este projeto é um sistema de e-commerce completo desenvolvido utilizando os princípios da Layered Architecture. O sistema
é projetado para gerenciar produtos e vendas, com funcionalidades que incluem controle de
estoque, gestão de descontos e geração de relatórios de vendas. Desenvolvido em Java, Spring Boot e Hibernate.
